# logic-simulator

## auto kompilacja

<code>npm run builder</code>

## uruchamianie electrona

<code>npm start</code>

## kompilowanie projektu

- package-win
- package-mac
- package-linux

## pisanie testów (są punktowanie w zadaniu)

1. stwórz skrypt importujący <b>../tester.js</b> w folderze tests
2. napisz testy
3. uruchom testy komendą <code>npm test -- tests/[nazwa skryptu].test.js</code>
4. nie usuwaj ich
